子模組程式碼

用cdbw17的名稱把https://github.com/scrum-1/cdbw17.git的倉儲加入為子模組

git submodule add -b gh-pages 子模組網址 子模組名稱
(例如:git submodule add -b gh-pages https://github.com/scrum-1/cdbw17.git cdbw17)

抓對應子模組的版本

git submodule update --init --recursive

更新子模組的資料,甚至更新子模組裡子模組的資料

git submodule foreach "(git checkout gh-pages; git pull)&"
